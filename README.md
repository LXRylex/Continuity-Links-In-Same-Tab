# Continuity - Experimwental Extension

As the title says, Let's you open links in same tab without opening a new tab

## Features:

Toggle ON/OFF
Mouse-Middle Click = Open links in new tab
CTRL+Click = Also open links in new tab

> [!WARNING]
> This Extension is **experimental**.
> Expect breaking/changes and unstable behavior. Only available at Firefox browsers.
